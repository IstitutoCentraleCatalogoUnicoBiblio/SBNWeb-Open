Descrizione

La funzione consente all'utente bibliotecario di modificare, per la biblioteca di appartenenza, i parametri della lettera d'ordine siano essi relativi alla modalità di attribuzione del numero del buono d'ordine (attraverso il bottone "Salva" di inserimento buono d'ordine) siano essi relativi al modello da usare come prospetto di stampa del buono d'ordine (attivabile on line tramite la pressione del pulsante "Stampa" da lista sintetica degli ordini o di dettaglio dell'ordine e in modalità differita da
  • Stampa Ordini)
    • . Essa consente in particolare:
    • modificare la numerazione del buono d'ordine(attribuzione automatica del sistema come progressivo o inserimento manuale dell'utente)
    • modificare l'intestazione, la formula introduttiva, il testo oggetto e i dati di fine stampa (firma) nel prospetto di stampa del buono d'ordine
    • indicare le aree ISBD che la biblioteca intende visualizzare, la presenza di un'immagine logo di biblioteca, la presenza del prezzo, dell'area relativa al numero e alla data di protocollazione, l'indicazione di ristampa, in caso di rinnovo l'indicazione della tipologia dell'ordine rinnovato (precedente/originario/assente), la presenza di un'immagine della firma in calce nel prospetto di stampa del buono d'ordine.

    Punti di ingresso

    1. Menu->Acquisizioni->Configurazione(tab Buono d'Ordine)
    2. Barra di navigazione

    Prerequisiti

    • Il bibliotecario deve avere superato con successo l'autenticazione e l'abilitazione alla funzione

    Modulo immissione dati

    Di seguito viene descritto il contenuto della schermata:

    • Impostazioni per la biblioteca
    • Dati di intestazione. Campo testuale che si presenta come riga di intestazione del buono
    • Formula introduttiva. Campo testuale che si presenta come formula introduttiva del buono ed è parametrizzabile per tipologia di ordine e di lingua.
    • Testo oggetto. Campo testuale che si presenta come oggetto del buono ed è parametrizzabile per tipologia di ordine e di lingua.
    • Dati fine stampa. Campo testuale che si presenta come dati di fine stampa (o di firma) del buono.
    • Lingua. Selezione da elenco con valori ammessi: "ITA": italiano, "ENG" inglese.
    • Tipo Ordine. Selezione da elenco con valori ammessi: "A-acquisto", "L-deposito legale", "D-dono", "V-visione trattenuta", "C-cambio", "R-Rilegatura"
    • Campo check (per consentire la selezione multipla)
    • N.ro buono d'ordine. Campo radio con valori ammessi: "Automatico", "Manuale"
    • Aree ISBD - Titolo. Campo radio con valori ammessi: "Si", "No"
    • Aree ISBD - Edizione. Campo radio con valori ammessi: "Si", "No"
    • Aree ISBD - Numerazione. Campo radio con valori ammessi: "Si", "No"
    • Aree ISBD - Pubblicazione. Campo radio con valori ammessi: "Si", "No"
    • Logo. Campo radio con valori ammessi: "Si", "No"
    • Immagine logo di biblioteca. Campo testuale in cui va specificato il percorso ed il nome del file di immagine locale selezionandolo con il bottone "Sfoglia" ed acquisendolo con il bottone 'Carica file selezionato'.
    • Prezzo. Campo radio con valori ammessi: "Si", "No"
    • Data. Campo radio con valori ammessi: "Si", "No"
    • Indicatore ordine rinnovato. Campo radio con valori ammessi per Rinnovo di: "originario", "precedente", "nessuno"
    • Firma digitale. Campo radio con valori ammessi: "Si", "No"
    • Immagine firma digitale. Campo testuale in cui va specificato il percorso ed il nome del file di immagine locale selezionandolo con il bottone "Sfoglia" ed acquisendolo con il bottone 'Carica file selezionato'.
    • Indicazione di ristampa. Campo radio con valori ammessi: "Si", "No"

    Valori di default

    • I valori di default possono variare in funzione della configurazione dei default gestiti dall'utente stesso. In caso l'utente non modifichi i default verranno presentati quelli di sistema. 

    Azioni utente

    • Selezione TAB folder relativo alla scelta della configurazione da effettuare (Buono d'Ordine, Ordine)
    • 'Codice Bibl. (Icona lista)'.  Elenco delle biblioteche affiliate a quella di appartenenza, con possibilità di scelta tramite selezione per consentire di operare per conto della biblioteca selezionata.
    • Bottone 'Salva': salva la configurazione di stampa per i buoni d'ordine della biblioteca e la modalità di attribuzione automatica/manuale del codice del buono d'ordine in fase di creazione.
    • Bottone 'Ripristina': consente di annullare tutte le modifiche effettuate sulla pagina prima del "Salva", recuperando le informazioni registrate in base dati.
    • Bottone 'Aggiungi intest./formula introduttiva/testo oggetto/fine stampa/ ': aggiunge una riga di testo nella sezione corrispondente.
    • Bottone 'Canc. intest./formula introduttiva/testo oggetto/fine stampa': cancella la riga di testo selezionata nella sezione corrispondente dal modulo di stampa.
    • Bottone 'Carica file selezionato': effettua l'upload ovvero l'acquisizione a sistema del file locale specificato con il bottone "Sfoglia" per la collocazione dell'immagine nel prospetto di stampa.
    • Bottone 'Sfoglia': consente la selezione di un'immagine locale da acquisire successivamente con il bottone "Carica file selezionato".

    Controlli del sistema

    Informazioni sulle tipologie di controlli e messaggistica
    • Invio diagnostico. Nel caso in cui, nel corso dell'esecuzione di qualunque passo, il sistema rilevi errori e/o incongruenze nei valori indicati dall'utente, o la ricerca non abbia esito positivo o il numero di elementi trovato sia troppo elevato per consentirne l'immediata visualizzazione, il sistema emetterà un opportuno diagnostico consentendo la correzione dei valori e la continuazione delle operazioni.
    • Il check sul campo "N.ro buono d'ordine" influenza la modalità di attribuzione del codice del buono d'ordine in fase di inserimento (automatico o manuale).
    • Al primo accesso nelle righe relative all'intestazione vengono proposti i dati memorizzati nell'anagrafica delle biblioteche (Denominazione, via, cap località) modificabili dell'utente. Per poter riaggiornare l'intestazione con i dati memorizzati nell'anagrafica delle biblioteche basterà cancellare tutte le righe di intestazione, salvare la configurazione e al nuovo accesso le righe saranno aggiornate con i dati voluti.
    • La prima riga di intestazione è riservata e viene considerata dal sistema come denominazione della biblioteca che sarà collocata in grassetto nel prospetto di stampa del buono d'ordine.
    • L'indicazione della presenza/assenza delle aree ISBD specificata nella configurazione, influenzano la descrizione dell'isbd che sarà stampata nel buono d'ordine.
